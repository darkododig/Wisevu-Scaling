Technical SEO
Fixes Master List

Full prioritized list of all technical SEO fixes. Quarterly audit surfaces all issues — fixes are worked through monthly by priority level.

Critical — Fix first
Important — Fix second
Low Priority — Fix third
📅

How This Works

A full site crawl is run once per quarter using Screaming Frog. All issues found are logged and prioritized into three tiers. Fixes are distributed across the three months of the quarter — critical issues in Month 1, important issues in Month 2, and low priority in Month 3. The SEO does not fix everything at once — the monthly allocation is determined by the client's program (Thrive, Ascend, or Embark).

Broken Links

  • Fix broken internal links
    Update URL or add a 301 redirect to the correct destination
    Critical
  • Fix broken external links
    Remove or replace with a working, relevant URL
    Important
📑

Indexation Issues

  • Resolve noindex tag errors
    Remove or correct unintentional noindex tags on pages that should be indexed
    Critical
  • Fix canonical tag errors
    Set the correct canonical URL to prevent duplicate content signals
    Critical
  • Resolve blocked pages in robots.txt
    Unblock pages that should be crawled, or confirm blocks are intentional
    Critical
🏥

E-E-A-T Signals (Healthcare Clients)

  • Verify author bylines are present on blog posts
    Google expects content authored by credible, named individuals for YMYL sites
    Important
  • Confirm About, Contact, and Privacy Policy pages are present and indexed
    These are trust signals for Google on healthcare and professional service sites
    Important
  • Flag pages making medical claims without proper attribution
    Add to content calendar for disclaimer addition during next content refresh
    Low
↪️

Redirect Issues

  • Clean redirect chains
    Collapse multi-hop redirects into a single 301 to preserve link equity
    Important
  • Fix redirect loops
    Identify pages redirecting to each other and break the loop
    Important
📋

Duplicate Content

  • Resolve duplicate page issues
    Set canonical, consolidate pages, or redirect as appropriate
    Important
  • Fix duplicate title tag issues
    Rewrite title tags to be unique across all pages
    Important
  • Fix duplicate meta description issues
    Rewrite meta descriptions to be unique across all pages
    Important
🗺️

Sitemap & Robots.txt

  • Fix any robots.txt syntax errors
    Confirm Googlebot is not accidentally blocked — critical if it is
    Critical
  • Remove 404 or redirected URLs from XML sitemap
    Sitemap should only contain canonical, indexable 200-status pages
    Important
  • Add missing important pages to XML sitemap
    Ensure all key service, location, and blog pages are included
    Important
🏷️

Title Tags & Meta

  • Fix missing title tags
    Every indexable page must have a unique, keyword-optimized title tag
    Critical
  • Fix missing meta descriptions
    Write unique meta descriptions for pages missing them
    Critical
  • Fix title tags outside 30–60 character range
    Rewrite titles that are too short (weak) or too long (truncated in SERPs)
    Important
⚙️

Schema Markup

  • Fix schema errors flagged in GSC
    Resolve invalid or incomplete structured data flagged in Google Search Console
    Important
  • Add missing LocalBusiness schema
    Ensure all location and contact pages have correct LocalBusiness structured data
    Important
  • Fix invalid schema implementations
    Validate using Google Rich Results Test and correct any errors
    Important
⚡

Core Web Vitals & Page Speed

  • Flag pages with poor LCP, INP, or CLS scores
    Log the page URL and failing metric in the master sheet so it can be addressed in a future on-page SEO update
    Important
  • Flag oversized images for compression
    Log in master sheet for developer or content team to action
    Low
📱

Mobile Usability

  • Flag mobile usability errors in GSC
    Tap target sizing, viewport issues, content wider than screen — escalate to developer
    Important
  • Verify previously flagged mobile issues are resolved
    Re-test after developer fixes and confirm in GSC
    Important
🔀

Internal Linking Structure

  • Identify orphaned pages
    Pages with no internal links pointing to them — add contextual links from relevant pages
    Important
  • Identify pages with only one internal link
    Flag for the On-Page SEO update task to add additional internal links
    Low
🔍

Google Search Console Alerts

  • Check GSC for manual penalty notices
    No action required unless a penalty is found — if one exists, escalate to team lead immediately
    Critical
  • Resolve crawl anomalies or coverage warnings
    Address submitted URLs marked as noindex, excluded, or discovered but not indexed
    Important
  • Submit updated sitemap if significant changes were made
    Resubmit via GSC after major content additions or structural changes
    Low
🔒

HTTPS & Security

  • Flag any HTTP pages that should be HTTPS
    Mixed content warnings — escalate to developer immediately
    Critical
  • Fix mixed content warnings
    Insecure resources (images, scripts) loading on secure pages
    Critical
  • Verify SSL certificate is valid and not expiring within 30 days
    Alert client and developer immediately if expiry is imminent
    Critical
🖼️

Image Optimization

  • Fix missing image alt text attributes
    Add descriptive, keyword-relevant alt text to images missing it
    Low
  • Flag oversized images for compression
    Log in master sheet for developer or content team to action — do not fix directly
    Low

Keyword Cannibalization

  • Identify keyword cannibalization
    Two or more pages competing for the same keyword — recommend consolidation, redirect, or differentiation
    Low

Crawl Budget

  • Review crawl stats in GSC for crawl budget waste
    Check if Googlebot is crawling low-value pages (tag pages, filtered URLs, thank-you pages)
    Low
  • Noindex low-value pages consuming crawl budget
    Tag pages, parameter URLs, pagination, thank-you pages — confirm with team lead before noindexing
    Low
📊

Analytics & Tracking

  • Verify GA and GSC are installed and tracking on all pages
    Flag any pages where tracking code is missing — escalate to developer
    Critical
  • Check for duplicate GA tags causing inflated session data
    Multiple GA instances fire on the same page — escalate to developer to remove duplicate
    Important
📍

Local SEO Technical

  • Verify NAP is consistent and crawlable on all location pages
    Name, Address, Phone must match GBP and citations exactly — flag discrepancies to Local SEO team
    Important
  • Check Google Maps embed is present on contact and location pages
    Escalate to developer if missing
    Low
  • Ensure location pages are not accidentally blocked from indexing
    Check robots.txt and noindex tags on all location and service area pages
    Critical

Open Graph & Social Tags

  • Verify OG title, description, and image are set on key pages
    Homepage, service pages, and blog posts should all have complete Open Graph tags
    Low
  • Fix missing or broken OG image tags
    Missing OG images result in poor-looking social shares — escalate to developer
    Low
