Department Overview

The Link Building Department is responsible for acquiring high-quality backlinks that increase each client's domain authority and organic rankings. All link building is manual — no PBNs, no link farms, no shortcuts. Each link builder manages 10 clients and is accountable for delivering 20–25 links per month across their entire client portfolio, using PitchBox for outreach management, Ahrefs for prospecting, and Hunter.io for contact discovery. The two primary tactics are guest posts and HARO / PR placements.

Link Building Tactics

Two core tactics are used for all client link building. Each requires a different workflow, timeline, and skill set.

✍️

Guest Posts

Primary tactic — manual outreach to relevant blogs and publications

  • Use Ahrefs to identify relevant blogs and publications in the client's niche (e.g. dental, health, local business) with DR 30+ and real organic traffic
  • Filter out link farms, PBNs, and sites that clearly sell links en masse
  • Find editor or contributor contact details using Hunter.io
  • Build and load outreach sequences in PitchBox — personalized pitch per site, not a template blast
  • Pitch a guest post topic relevant to both the publication's audience and the client's services
  • Once accepted, write or brief the guest post (600–900 words) with one contextual link back to the client's target page
  • Use natural, varied anchor text — avoid exact-match anchors on every link
  • Follow up once if no response after 5–7 business days
  • Track accepted, rejected, and pending pitches in PitchBox pipeline
  • Confirm the link is live before logging it in the monthly report
📣

HARO / PR Links

Reactive outreach — responding to journalist and media requests

  • Monitor HARO (Help a Reporter Out) or equivalent PR platforms (Connectively, Qwoted) daily for relevant queries
  • Filter queries by relevance to the client's niche and expertise (e.g. a dentist answering dental health questions)
  • Draft concise, expert responses on behalf of the client — 150–300 words, specific and quotable
  • Include the client's name, title, and practice name in every response for attribution
  • Submit responses promptly — HARO queries have tight deadlines (often same-day)
  • Track all submitted responses and publication outcomes in PitchBox or a dedicated tracker
  • Follow up with the journalist if publication is expected but not confirmed within 2 weeks
  • HARO links typically come from high-DR news and media sites — prioritize these when available
  • Log confirmed live links with the publication name, DR, and link URL in the monthly report

PitchBox Campaign Management

PitchBox is the central platform for all outreach. Each client has their own campaign. Here's how campaigns are set up and managed.

Campaign Setup (New Clients)

  • Create a new PitchBox campaign for each client — named clearly by client and tactic (e.g. "Client Name — Guest Posts")
  • Import prospect list from Ahrefs export — filter for DR 30+, relevant niche, organic traffic above 500/mo
  • Enrich contact data using Hunter.io — find editor or contributor email addresses
  • Write 2–3 personalized email templates per campaign — vary subject lines and opening hooks
  • Set follow-up sequence: initial pitch → follow-up at day 6 → close at day 12
  • Review all templates before activating — no generic or spammy language

Ongoing Campaign Management

  • Check PitchBox daily for replies, acceptances, and rejections
  • Move accepted pitches to "In Progress" and begin content or response drafting immediately
  • Log rejections and unsubscribes — do not re-pitch the same contact
  • Replenish prospect lists monthly — add 30–50 new prospects per client per month to keep pipeline healthy
  • Monitor email deliverability — watch for bounce rates and spam folder placements
  • Archive completed campaigns and start fresh quarterly to avoid list fatigue

Prospect Research Standards

  • Use Ahrefs Site Explorer to vet every prospect before outreach
  • Minimum DR 30 — exceptions only for highly relevant niche sites with strong editorial standards
  • Must have real organic traffic — reject sites with zero or suspiciously flat traffic graphs
  • Check for link spam indicators: excessive outbound links, thin content, unrelated niches
  • Prioritize sites that have published similar guest posts recently — signals active editorial
  • Never prospect sites already linking to the client — check existing backlink profile first

Anchor Text Strategy

  • Vary anchor text across every link built — never repeat the same anchor for the same client
  • Anchor text distribution target: 40% branded, 30% partial match, 20% natural/generic, 10% exact match
  • Review the client's existing backlink profile in Ahrefs before deciding anchor text — fill gaps, don't duplicate
  • Avoid over-optimized anchors like "best dentist Toronto" on every link — triggers Penguin risk
  • Document every anchor used in the monthly link report
  • Flag to On-Page SEO team if target URLs need to be updated based on ranking priorities

Link Quality Standards

Every link built must meet these standards before being logged. Quality over quantity — a few strong links outperform dozens of weak ones.

✅ Accepted links

What qualifies

  • DR 30 or above (Ahrefs Domain Rating)
  • Real organic traffic — verified in Ahrefs Site Explorer
  • Topically relevant to the client's industry or services
  • Contextual placement — link is within body content, not footer or sidebar
  • Dofollow link (nofollow acceptable only for high-DR media/PR placements)
  • Site has editorial standards — not a link farm or paid link directory
  • Link points to a relevant page on the client's site, not just the homepage
⚠️ Rejected links

What does not qualify

  • DR below 30 unless exceptional editorial relevance
  • Zero or fake organic traffic
  • Sites with excessive outbound links on every page
  • Private blog networks (PBNs) — immediate disqualification
  • Unrelated niches (e.g. a gambling site linking to a dental clinic)
  • Sitewide links in footers, widgets, or navigation
  • Paid links disclosed as sponsored without nofollow attribute

Monthly Workflow

Link building is an ongoing pipeline — not a one-time task. The workflow below keeps outreach consistent and links flowing every month.

1
Pipeline Review & Prospect Refresh (Week 1)
Review all active PitchBox campaigns. Check reply rates, bounce rates, and pipeline health. Add 30–50 new prospects per client using Ahrefs — keeping the pipeline full is the most important ongoing task. Enrich new prospects with Hunter.io contact data.
2
HARO Monitoring (Daily — all 4 weeks)
Check HARO / Connectively every morning for relevant queries across all 10 clients. Draft and submit expert responses for matching queries on the same day — HARO is time-sensitive. Log all submissions in the tracker. This runs in parallel with everything else throughout the month.
3
Outreach Activation & Follow-Ups (Week 1–2)
Send new outreach batches for guest post campaigns. Follow up on pitches from the prior month that haven't received a response. Move accepted pitches to active status and begin drafting or commissioning guest post content immediately.
4
Guest Post Content & Submission (Week 2–3)
Write or brief guest post content for all accepted pitches. Each post is 600–900 words, genuinely useful to the publication's audience, with one natural contextual link back to the client's target page. Submit to the editor and track submission status in PitchBox.
5
Link Verification (Week 3–4)
Confirm all expected links are live. Check each URL directly and verify in Ahrefs that the link has been indexed. If a promised link hasn't gone live, follow up with the editor. Never log a link as delivered until it is confirmed live and crawlable.
6
Monthly Link Report (Week 4)
Compile the monthly link report for all 10 clients. Each report includes: live links acquired, referring domain, DR, anchor text used, target URL, and link type (guest post or HARO). Submit to Account Managers for client delivery. Flag any clients who fell short of their target with a reason and recovery plan.
